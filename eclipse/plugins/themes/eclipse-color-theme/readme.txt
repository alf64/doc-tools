eclipse-color-theme-update-site.zip - color theme plugin for Eclipse
Install it manually by unzipping the contents and copy over the files from 'features' and 'plugins' directories into
the 'features' and 'plugins' directories under your Eclipse installation directory. Restart the Eclipse.
Upon successful installation you shall find Color Theme under Preferences -> General -> Appearance -> Color Theme where you can set
your favourite theme (mine is Vibrant Ink).

Warning! This plugin do not work on STM32CubeIDE
